Culture Ship Name Picker
===============================

version number: 0.0.1
author: Signal

Overview
--------

 Get random ship names from Ian Banks' Culture novels

Installation / Usage
--------------------

To install use pip:

    $ pip install shipnames


Or clone the repo:

    $ git clone https://github.com/signalrock/shipnames.git
    $ python setup.py install
